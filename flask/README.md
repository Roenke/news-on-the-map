# Простой web-интерфейс для приложения

### Зависимости:
- python3
- sqlite3
- flask
- sqlalchemy
- flask-sqlalchemy
- sqlalchemy-migrate

### Установить все зависимости:
```
apt-get install sqlite3 python3 python3-flask python3-sqlalchemy && pip3 install flask-sqlalchemy sqlalchemy-migrate
```

### Запуск:
```
./run.py
```
